### NodeLink

**Contribution:** Added full Apple Music source support and native lyrics retrieval through reverse-engineered Apple Music APIs.

Key additions:

* Apple Music song, album, playlist, and artist support

* Apple Music search integration

* Native lyrics retrieval

* Pagination support for large collections

* Artwork parsing improvements

* Lavalink-compatible track encoding

* Setup locking and token validation

* Metadata extraction and fallback mirroring

* Repository: https://github.com/PerformanC/NodeLink

* Pull Request: https://github.com/PerformanC/NodeLink/pull/116
