### NodeLink
**Implemented full Apple Music support** through reverse-engineered Apple Music APIs.

#### ✨ Highlights
- Apple Music songs, albums, playlists and artists
- Apple Music search integration
- Native lyrics retrieval
- Collection pagination
- Artwork parsing improvements
- Lavalink-compatible track encoding
- Token validation & setup locking
- Metadata extraction & mirroring fallback
